# Student_Management_System-_-Python-

A simple Student Management System built using Python.

## Features

- Add Student
  
- View Students
  
- Search Student
  
- Update Student
  
- Delete Student
  
- Data stored using JSON file

## Technologies Used

- Python
  
- File Handling
  
- JSON

## How to Run

1. Install Python
   
2. Run the file:

python student.py
